## Purpose

QZN functions as a closed-loop arcade utility token. It is:

- Earned through skill-based gameplay, tournaments, and creation
- Spent on arcade features, access, and protocol actions
- Permanently burned through many usage paths

QZN does not represent ownership, profit rights, governance control, or financial claims.

---

## Earning QZN

QZN may be earned through verifiable contribution:

- Skill-based gameplay and achievements
- Tournament participation and placement
- Game and content creation
- Community and ecosystem participation
- Seasonal arcade events

Anti-abuse mechanisms include:
- Daily earning caps
- Skill-weighted rewards
- Vesting delays

---

## Spending & Burn Mechanics

QZN is designed to be consumed through use.

Examples of burn sinks:
- Cosmetic items and visual effects
- Name changes and badges
- Game publishing and verification
- Arcade signaling actions

Some spending paths recycle QZN into event pools and builder rewards, while others permanently remove supply.

---

## Locking & Access (No Yield)

Locking QZN provides access and permissions within the arcade ecosystem.

- No yield
- No profit share
- No financial returns

Higher lock tiers unlock features such as:
- Early access
- Tournament hosting
- Creator tools
- Prestige status

---

## Protocol Rent

Arcade applications built on Qubic pay protocol rent in QZN for actions such as:

- Game publishing
- Tournament hosting
- Marketplace activity
- NFT minting

This establishes QZN as the coordination unit of the arcade layer.

---

## Transparency

The arcade ecosystem operates on a weekly epoch cycle that reports:

- QZN burned
- Events funded
- Creators rewarded
- Games launched
