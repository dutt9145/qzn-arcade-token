# Contributing to QZN

Thank you for your interest in contributing to the QZN arcade ecosystem.

## How to Contribute
- Open an issue for bugs, questions, or suggestions
- Submit pull requests for documentation improvements
- Propose arcade features or tooling improvements

## Guidelines
- Keep contributions non-financial and arcade-focused
- No investment, pricing, or speculative language
- Follow Qubic ecosystem standards

## Review Process
All contributions are reviewed by maintainers before merging.
